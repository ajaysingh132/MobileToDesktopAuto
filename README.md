# MobileToDesktopAuto
मोबाइल में डेस्कटॉप अनुभव के लिए इस स्क्रिप्ट का उपयोग करें।

### [👉 यहाँ क्लिक करके इंस्टॉल करें (Install Now)](https://raw.githubusercontent.com/ajaysingh132/MobileToDesktopAuto/main/script.user.js)

## निर्देश:
1. **Kiwi Browser** या **Firefox** इंस्टॉल करें।
2. ब्राउज़र में **Tampermonkey** एक्सटेंशन जोड़ें।
3. ऊपर दिए गए "Install Now" लिंक पर क्लिक करें।
4. स्क्रिप्ट को **Install** करें और पेज रिफ्रेश करें।
